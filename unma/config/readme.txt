disini terdapat beberapa file konfigurasi
- db_config.json
    contoh:
    {
        "DATABASE": <database url connection>,
        "DATABASE_NAME": <database name>
    }

- fcm_config.json
    contoh:
    {
        "FCM_SERVER_KEY": <random firebase server key>
    }

- path_config.json
    contoh:
    {
        "UPLOAD_FOLDER": <path to upload folder>
    }

file diatas akan digenerate secara otomatis oleh script util.py
